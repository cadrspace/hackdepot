# HackDepot

Hackspace inventory platform

## Installation & requirements

Ruby >= 2.2.2
MongoDB >= 3.x

You need to install rvm http://rvm.io
Install ruby through RVM
go to project root

change database credentials in **config/mongoid.yml**

```
bundle install
```

```
rake db:setup
```

Start server 

```
rails s 
```
